Question 1 (25 points) - jQuery + objet simple JavaScript

Contexte : Vous offrez aux utilisateurs de votre application la possibilité de sélectionner l'option de livraison qui leur convient au moment de payer. Lorsqu'ils sélectionnent une des options, les informations de paiement sont mises à jour pour refléter le prix de livraison et le total de la commande.

Votre travail est de répliquer le comportement présenté dans l'aperçu vidéo.

Note : Tout le travail se fait dans le fichier 'app.js', sous le code fourni. Les autres fichiers doivent être remis dans leur état initial. 

Informations utiles :
- La classe .bg-info permet de mettre les rangées de la table de gauche en surbrillance
- Ces méthodes jQuery peuvent vous être utiles:
  - .html()
  - .addClass()
  - .removeClass()
- L'événement à observer est un "click" sur les boutons de sélection.
